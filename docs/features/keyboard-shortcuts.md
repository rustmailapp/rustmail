# Keyboard Shortcuts

RustMail's web UI supports vim-style keyboard navigation.

| Key | Action |
|-----|--------|
| `j` | Select next message |
| `k` | Select previous message |
| `d` | Delete selected message |
| `D` | Delete all messages |
| `/` | Focus search bar |
| `Esc` | Close message detail panel |

Shortcuts are active when no input field is focused.
