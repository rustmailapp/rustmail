# API Reference

Base path: `/api/v1`. All responses are JSON. All IDs are [ULIDs](https://github.com/ulid/spec) (time-sortable, globally unique).

The full OpenAPI 3.1 spec is also served at runtime at `GET /api/v1/openapi.yaml`.
